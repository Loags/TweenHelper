# TweenHelper demos

This sample contains the 2D UI showcase and the full 3D preset showcase. Both scenes include the reset-audit runner used to validate preset state restoration.

Before opening the scenes, import TextMesh Pro Essential Resources. The 3D scene materials are authored for the Universal Render Pipeline; the 2D scene and TweenHelper runtime are render-pipeline independent.

The demos do not require the Input System package. Keyboard and fly-camera shortcuts are available when the legacy Input Manager is enabled, and the public component/context-menu controls remain available otherwise.
