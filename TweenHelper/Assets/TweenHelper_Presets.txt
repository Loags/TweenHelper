────────────────────────────────────────
About TweenHelper
────────────────────────────────────────

TweenHelper is a high-level animation facade built on DOTween for Unity.
It provides a fluent builder API, named presets with auto-registration,
sequence composition, and automatic cleanup via GameObject linking.

Builder Capabilities:
  Move, Rotate, Scale, Fade, Sequence (Then/With), Async/Await

────────────────────────────────────────
Architecture Overview
────────────────────────────────────────

Fluent Builder    — transform.Tween().Move(...).WithEase(...).Play()
Preset System     — [AutoRegisterPreset] attribute + TweenPresetRegistry auto-discovery
TweenHandle       — Wrapper with control (Pause/Resume/Kill), status, and async/await
TweenOptions      — Value-type struct with nullable fields for per-call overrides
Auto-Cleanup      — SetLink(gameObject) + TweenLifecycleTracker fallback
Global Defaults   — TweenHelperSettings singleton ScriptableObject

[Flicker]
Description: Randomized alpha flicker effect

Produces randomized alpha flickering using Perlin noise, snapping to full opacity on completion.

Uses DOVirtual.Float(0, 1, dur) with a per-invocation random seed to sample
Mathf.PerlinNoise(seed + t * 12, 0) each frame, directly setting the target's alpha.
The 12x time multiplier produces rapid, organic fluctuations. On complete, alpha is
explicitly set to 1.0 to ensure a clean final state.

Type: One-shot effect | Default duration: 1.0s | Default ease: Linear (progress)

Easing override: Ease has minimal visible effect (controls progress, not noise).

Requires: A fadeable component.

Use cases: Torch/campfire flicker, electrical malfunction, glitch effect, dying light.

Usage: transform.Tween().Preset("Flicker").Play();

Strength override: Multiplies zigzag distance (default 1.0).
